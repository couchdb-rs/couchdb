Struct couchdb::path::DatabaseName [−] [src]

pub struct DatabaseName(_);

DatabaseName is a single URL path segment that specifies the name of a database.

For example, given the document path /db/_design/doc, the database name is db.

For more information about path-related types, see the module-level documentation.
